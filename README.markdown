DOIG.ME
=======
This repository contains the source code for the layout and content for the [doig.me](http://doig.me/) website. 

Installation
============
$ > cd doig.me.git/  
$ > virtualenv .  
$ > source bin/activate  
$ > pip install -e git://github.com/lakshmivyas/hyde.git#egg=hyde  
$ > ./bin/hyde -g -s site/ -d STATIC_DIRECTORY
