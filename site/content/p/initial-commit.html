At the moment, I'm hoping that this blog will host an array of interesting yet-to-be-written articles covering topics ranging from web development to indie music and everything in between. When creating this site I was looking for something that was simplistic to administrate but also technically interesting enough to tickle my engineering side. What I've settled upon is something I am very pleased with. The two most important tools in my toolbox (debatedly) are [Python](http://python.org) and [Git](http://gitscm.org). These are two technologies that I use daily and are my primary means of creating digital things. The pages you're able to view on this website are good 'ol fashioned static files. The files are generated using [Lakshmi Vyas'](http://ringce.com/blog) [Hyde - The Python Static Site Generator](http://github.com/lakshmivyas/hyde), written in the [Markdown](http://daringfireball.net/projects/markdown/) formatting syntax, and deployed to this server using a set of simple Git post-commit hooks. I've released the source code of this website under the MIT license, available on [Github](https://github.com/doigoid/doig.me). I'm planning on writing a tutorial of sorts, on how you can easily set up a staticly generated website yourself. Thanks for reading, I hope to see you back soon! {% endblock %}
